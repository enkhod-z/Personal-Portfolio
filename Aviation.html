Exploratory Data Analysis¶

For this notebook we will explore a dataset in class. As a reference point, this notebook goes over several great EDA principles: https://nbviewer.jupyter.org/github/PBPatil/Exploratory_Data_Analysis-Wine_Quality_Dataset/blob/master/winequality_white.ipynb

The purpose of this notebook is to explore the avaiation safety dataset from Kaggle (link).

The following features are in the dataset:

  • airline: Airline name
  • avail_seat_km_per_week: miles
